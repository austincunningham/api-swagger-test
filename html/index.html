get /apps/{appId}
Get app by appId (getAppById)
Retrieve all information for a single app including all child information

Path parameters

appId (required)
Path Parameter — The id for the app that needs to be fetched.

Return type

App

Example data

Content-Type: application/json
{
  "disabledMessage" : "disabledMessage",
  "appName" : "appName",
  "deployedVersions" : [ {
    "numOfClients" : 5,
    "disabledMessage" : "disabledMessage",
    "disabled" : false,
    "id" : 1,
    "version" : "version",
    "numOfAppStartups" : 5
  }, {
    "numOfClients" : 5,
    "disabledMessage" : "disabledMessage",
    "disabled" : false,
    "id" : 1,
    "version" : "version",
    "numOfAppStartups" : 5
  } ],
  "appId" : 6,
  "disabled" : false,
  "id" : 0
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

successful operation App

400

Invalid appId supplied

404

App not found
